# L1 主流程模板（内联子流程）

文件名: `FL-{abbr}-{NN}-{title}.md`（如 `FL-SIEGE-01-攻城战.md`）

**设计要求**: 一个文件包含 L1 主流程概览 + 所有子流程内联。子流程用 `##` 标题分隔，不单独拆分。文件超长（> 500行）时按阶段拆分为多个文件。

---

```markdown
# FL-{abbr}-{NN} {功能名称}

> **版本**: v{N} | **最后更新**: YYYY-MM-DD
> **归属功能**: [功能模块描述]
> **触发条件**: [什么触发此主流程开始]
> **结束条件**: [什么标志此主流程结束]
> **前置状态**: [系统/用户在流程开始前的状态]
> **后置状态**: [系统/用户在流程结束后的状态]
> **来源文档**: [PLAN.md](../PLAN.md) 中的原始文档引用

## 关键要求

- [全局性业务约束]
- [...]

## 验收标准

- [主流程完成的判定条件]
- [...]

## 主流程概览

```
FL-{abbr}-{NN} {功能名称}
├── Phase A: {阶段名称}  →  FL-{abbr}-{NN}-01 {子流程名}
├── Phase B: {阶段名称}  →  FL-{abbr}-{NN}-02 {子流程名}
├── ...
└── Phase N: {阶段名称}  →  FL-{abbr}-{NN}-{NN} {子流程名}
